2. 相对位置 aware self-attention

  • 提出：Shaw P, Uszkoreit J, Vaswani A. Self-attention with relative position representations[J]. arXiv preprint arXiv:1803.02155, 2018.

  • 原理

    通过在 K、V 表示上加可学习的相对位置表示，在 attention 计算中引入相对位置信息，提高模型性能。并将相对位置限制为 [-k, k]，假设超过一定距离后，精确的相对位置信息是无用的；还使模型能够泛化到训练期间未见过的序列长度。通过跨头、跨序列共享相对位置表示，降低增加的空间复杂度。

3.1. 远程衰减

随着相对距离增加，q\mathbf{q}q、k\mathbf{k}k 的内积震荡衰减。

python 代码 
import numpy as np
import matplotlib.pyplot as plt


def rotate(v, theta):
    """
    Rotate vector v by theta radians.
    """
    rotation_matrix = np.array([[np.cos(theta), -np.sin(theta)], [np.sin(theta), np.cos(theta)]])
    return np.dot(rotation_matrix, v)


def calculate_similarity(q, k, position_q, position_k):
    """
    Calculate the similarity of RoPE-encoded vectors.
    """
    theta_q = np.array([10000**(-2 * i / dim) * position_q for i in range(len(q))])
    theta_k = np.array([10000**(-2 * i / dim) * position_k for i in range(len(k))])
    q_rope = [rotate(q[i], theta_q[i]) for i in range(len(q))]
    k_rope = [rotate(k[i], theta_k[i]) for i in range(len(k))]
    q_rope = np.concatenate(q_rope, axis=0)
    k_rope = np.concatenate(k_rope, axis=0)

    return np.dot(q_rope, k_rope.T)


dim = 128
q = np.random.rand(int(dim / 2), 2)  # dim-dimension vector divided into dim / 2 2D sub-vectors
k = np.random.rand(int(dim / 2), 2)

# Calculate similarity for different positions
position_q = 128
positions_k = range(256)
similarities = [calculate_similarity(q, k, position_q, position_k) for position_k in positions_k]

plt.plot([x - position_q for x in positions_k], similarities)
plt.xlabel('position_k - position_q')
plt.ylabel('attention score')
plt.title('Similarity decay for RoPE')
plt.show()
plt.savefig('rope.png')

4. ALiBi

  • 提出：Press O, Smith N A, Lewis M. Train short, test long: Attention with linear biases enables input length extrapolation[J]. arXiv preprint arXiv:2108.12409, 2021.

  • 目标：长度外推。训练时采用短序列，推理时对更长的序列进行有效推断。

  • 原理：不向 word embedding 加 position embedding，而是在 QK attention score 上加一个与距离成比例的线性惩罚来实现外推。

    softmax⁡(qiK⊤+m⋅[−(i−1),…,−2,−1,0])\operatorname{softmax}\left(q_i K^{\top}+m \cdot[-(i-1), \ldots,-2,-1,0]\right) softmax(qi​K⊤+m⋅[−(i−1),…,−2,−1,0])

    其中，mmm 为头特定的斜率超参数。

  • 斜率选择：对于 nnn 个头的模型，斜率值以 2−8n2^{\frac{-8}{n}}2n−8​ 为起始值、ratio。如8个头的模型，采用 121,122,…,128\frac{1}{2^1}, \frac{1}{2^2}, \ldots, \frac{1}{2^8}211​,221​,…,281​；16个头的模型，采用 120.5,121,121.5,…,128\frac{1}{2^{0.5}}, \frac{1}{2^1}, \frac{1}{2^{1.5}}, \ldots, \frac{1}{2^8}20.51​,211​,21.51​,…,281​。

  • 相关工作：正弦位置 embedding、RoPE 长度外推能力有限；相对位置编码计算复杂度高、需要额外的内存。效果对比详见论文表 2-4。且 ALiBi 无额外的运行时开销。

  • 效果：可以实现与直接在长序列上训练、采用正弦位置编码的模型相当的困惑度，训练速度更快，内存需求更小。长度外推性能在训练序列长度两倍左右达到峰值，详见论文图 6。不进行长度外推时，可实现更低或相似的困惑度。

  • 采用滑动窗口测试，外推到更长的序列时性能保持平稳或下降（区别于非重叠窗口测试），表明 ALiBi 并不能利用更长的上下文进行更准确的预测。
