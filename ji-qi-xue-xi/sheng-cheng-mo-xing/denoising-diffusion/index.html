denoising diffusion

机器学习 生成模型
发布日期:   2024-06-30
更新日期:   2024-07-11
  • 论文：Ho J, Jain A, Abbeel P. Denoising diffusion probabilistic models[J]. Advances in neural information processing systems, 2020, 33: 6840-6851.
  • 核心思想
    • 将无条件生成近似为马尔可夫链过程，转移概率为条件高斯，反向过程由噪声 p(xT)=N(xT;0,I)p\left(\mathbf{x}_T\right) = \mathcal{N}\left(\mathbf{x}_T ; \mathbf{0}, \mathbf{I}\right)p(xT​)=N(xT​;0,I) 渐进式地生成数据，正向过程为逐步向数据加高斯噪声直到信号被破坏。
    • 将正向过程的分布参数固定为常数超参数，通过重参数化 Eq. (8-1)、Eq. (11)、Eq. (7-3)，得到变分界 Eq. (12)。进一步简化为 Eq. (14)，类似于对较小的 ttt 降低了 loss 权重，有利于采样质量。
    • 对于图像离散数据，采用 Eq. (13) 所示的离散 decoder，使得在生成最终数据时不需要加噪。
    • 训练、采样流程

$$ q\left(\mathbf{x}_{t-1} \mid \mathbf{x}_t, \mathbf{x}0\right)=\mathcal{N}\left(\mathbf{x}{t-1} ; \tilde{\boldsymbol{\mu}}_t\left(\mathbf{x}_t, \mathbf{x}_0\right), \tilde{\beta}_t \mathbf{I}\right) \tag{6}$$
$$ \tilde{\boldsymbol{\mu}}_t\left(\mathbf{x}_t, \mathbf{x}0\right):=\frac{\sqrt{\bar{\alpha}{t-1}} \beta_t}{1-\bar{\alpha}_t} \mathbf{x}0+\frac{\sqrt{\alpha_t}\left(1-\bar{\alpha}{t-1}\right)}{1-\bar{\alpha}_t} \mathbf{x}_t \tag{7-1}$$

\tilde{\beta}_t:=\frac{1-\bar{\alpha}_{t-1}}{1-\bar{\alpha}_t} \beta_t \tag{7-2}$$</font> 其中，高斯分布之间的 KL 散度可以直接计算。 # Forward process 相较于通过重参数化学习 $\beta_t$，本文将其固定为常数超参数，因此近似后验 $q$ 没有可学习的参数，$L_T$ 为常数，训练时可忽略。 # Reverse process 令 <span id="Eq7-3">$$\boldsymbol{\Sigma}_\theta\left(\mathbf{x}_t, t\right)=\sigma_t^2 \mathbf{I} \tag{7-3}$$</span> 为时间相关的常数。 则 $$ L_{t-1}=\mathbb{E}_q\left[\frac{1}{2 \sigma_t^2}\left\|\tilde{\boldsymbol{\mu}}_t\left(\mathbf{x}_t, \mathbf{x}_0\right)-\boldsymbol{\mu}_\theta\left(\mathbf{x}_t, t\right)\right\|^2\right]+C \tag{8}

$$ L_{\text {simple }}(\theta):=\mathbb{E}_{t, \mathbf{x}0, \boldsymbol{\epsilon}}\left[\left|\boldsymbol{\epsilon}-\boldsymbol{\epsilon}\theta\left(\sqrt{\bar{\alpha}_t} \mathbf{x}_0+\sqrt{1-\bar{\alpha}_t} \boldsymbol{\epsilon}, t\right)\right|^2\right] \tag{14}$$

3. L0L_0L0​、离散 decoder

  • 将图像数据 [0，255] 缩放到 [-1, 1]。

  • 离散解码器/离散 likelihood

    $$ p_\theta\left(\mathbf{x}0 \mid \mathbf{x}1\right) =\prod{i=1}^D \int{\delta_{-}\left(x_0i\right)}{\delta_{+}\left(x_0^i\right)} \mathcal{N}\left(x ; \mu_\theta^i\left(\mathbf{x}1, 1\right), \sigma_1^2\right) dx \
    \delta    {+}(x) =\left{\begin{array}{ll}
    \infty & \text { if } x=1 \
    x+\frac{1}{255} & \text { if } x<1
    \end{array} \quad \delta_{-}(x)= \begin{cases}-\infty & \text { if } x=-1 \
    x-\frac{1}{255} & \text { if } x>-1\end{cases} \right.\tag{13}$$

    其中，DDD 表示数据维度，iii 表示坐标。可以替换为更强大的解码器，如条件自回归模型。

    使得对离散数据是无损编码，且在生成最终数据时不需要加噪。

6. 实验

实验 结果 结论
采样质量 采用真实变分界，NLL 更好；采用简化版本，采样质量更好。
反向过程参数化、训练目标 1. 预测 μ~\tilde{\boldsymbol{\mu}}μ~​ 时，采用 MSE loss（类似于Eq. (14)）训练不稳定，可采用真实变分界；
2. 相较于固定反向过程方差，学习方差导致样本质量较差或训练不稳定；
3. 相较于预测 μ~\tilde{\boldsymbol{\mu}}μ~​，预测 ϵ\boldsymbol{\epsilon}ϵ，采用真实变分界、固定方差时性能相当，采用简化版本时性能更好。
