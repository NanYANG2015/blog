wav2vec 2.0

音频 音频表示 论文
发布日期:   2024-10-15
更新日期:   2024-11-01
  • Baevski A, Zhou Y, Mohamed A, et al. wav2vec 2.0: A framework for self-supervised learning of speech representations[J]. Advances in neural information processing systems, 2020, 33: 12449-12460.
  • 作者：来自 Facebook。
  • https://github.com/facebookresearch/fairseq
  • 创新点：连续表示包含更多信息，用于上下文表示；对比 loss 采用量化之后的表示，连续表示包含说话人、背景声音等信息，使得对比任务更容易，但不利于学习用于语音识别的通用表示。
  • 应用：提高低资源语种、仅少量标注数据的任务的性能。

  • CNN encoder：输入波形，输出约为 20ms/帧，感受野约为 25ms。

  • Transformer

    • 相对位置 embedding：采用类似 [37] 的1个卷积层。
    • mask 策略：以概率 \(p=0.065\) 随机采样 mask 段的起始索引，mask 连续的 \(M=10\) 帧，可能重叠。导致约 49% 的时刻被 mask，平均 mask 长度约为 299ms。消融实验详见表 5。
    • mask 表示：全部替换为1个可学习的特征向量。

  • 量化：product quantization，\(G=2\) 个码本，码本大小为 \(V=320\)。采用 hard (argmax) Gumbel softmax，将 encoder 输出 \(z\) 映射到 \(\mathbf{l} \in \mathbb{R}^{G \times V}\)，选择第 \(g\) 个码本中的第 \(v\) 个值的概率为 \[p_{g, v}=\frac{\exp \left(l_{g, v}+n_v\right) / \tau}{\sum_{k=1}^V \exp \left(l_{g, k}+n_k\right) / \tau}\] 其中 \(n=-\log (-\log (u))\)，\(u\) 从 \(\mathcal{U}(0,1)\) 中均匀采样，为服从 Gumbel 分布的噪声。Gumbel softmax 接近 one-hot 向量。采用 straight-through estimator，前向使用 argmax 选择码字，反向使用 Gumbel softmax 的梯度。训练时，\(\tau\) 从 2 逐步退火到 0.5（BASE）/ 0.1（LARGE）。

  • 对比 loss + 码本 diversity loss

    \[ \begin{align} \mathcal{L} &=\mathcal{L}_m+\alpha \mathcal{L}_d \\ \mathcal{L}_m &=-\log \frac{\exp \left(\operatorname{sim}\left(\mathbf{c}_t, \mathbf{q}_t\right) / \kappa\right)}{\sum_{\tilde{\mathbf{q}} \sim \mathbf{Q}_t} \exp \left(\operatorname{sim}\left(\mathbf{c}_t, \tilde{\mathbf{q}}\right) / \kappa\right)} \\ \mathcal{L}_d &=\frac{1}{G V} \sum_{g=1}^G-H\left(\bar{p}_g\right)=\frac{1}{G V} \sum_{g=1}^G \sum_{v=1}^V \bar{p}_{g, v} \log \bar{p}_{g, v} \end{align}\]

    其中，\(\alpha=0.1\)，\(\kappa=0.1\)，\(\operatorname{sim}(\mathbf{a}, \mathbf{b})\) 为余弦相似度，\(\bar{p}_{g, v}\) 为 batch 内所有句子的 \(l_{g, v}\) 的平均 softmax 分布，最大化其熵。对比 loss 中干扰项从同一句其它被 mask 的时刻均匀采样 100 个。diversity loss 激励模型训练时平等地使用每个码本中的所有条目。

  • 结果分析：预训练模型得到的语音表示可以很容易地用于识别特定的声音，进一步微调可以使其对应到正确的单词拼写。

* LV-60k: LibriVox
