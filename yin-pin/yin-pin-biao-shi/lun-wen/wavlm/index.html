  • Chen S, Wang C, Chen Z, et al. Wavlm: Large-scale self-supervised pre-training for full stack speech processing[J]. IEEE Journal of Selected Topics in Signal Processing, 2022, 16(6): 1505-1518.
  • 作者：来自微软
  • https://github.com/microsoft/unilm/tree/master/wavlm
  • 创新点
    • 模拟带噪、重叠语音，在掩码区域预测原始语音类 HuBERT 获取的离散标签，对说话人相关的下游任务有显著提升。
    • 采用 gated relative position bias，相较于卷积相对位置 embedding，允许根据当前语音内容自适应地调整，对内容相关的下游任务有显著提升。
    • 训练数据：相较于之前的模型无监督语音主要为有声读物，与现实场景不匹配，采用 Libri-Light + GigaSpeech + VoxPopuli，共 94kh 英文语音。
    • 在 SUPERB、说话人验证、语音分离、说话人分离任务上实现了 SOTA 性能，ASR 性能与 wav2vec 2.0、HuBERT 相当。

2. gated relative position bias

\[ \begin{aligned} a_{ij} & \propto \exp \left\{\frac{\mathbf{q}_i \cdot \mathbf{k}_j}{\sqrt{d_k}}+r_{i-j}\right\} \\ g_i^{(\text {update })}, g_i^{\text {(reset) }} &= \sigma\left(\mathbf{q}_i \cdot \mathbf{u}\right), \sigma\left(\mathbf{q}_i \cdot \mathbf{w}\right) \\ \tilde{r}_{i-j} &= w g_i^{\text {(reset) }} d_{i-j} \\ r_{i-j} &= d_{i-j}+g_i^{\text {(update) }} d_{i-j}+\left(1-g_i^{\text {(update) }}\right) \tilde{r}_{i-j} \\ d_{|i-j|} &= \begin{cases}|i-j|, & |i-j|<\frac{n}{4} \\ \left\lfloor\frac{n}{4}\left(\frac{\log (|i-j|)-\log \left(\frac{n}{4}\right)}{\log (m)-\log \left(\frac{n}{4}\right)}+1\right)\right\rfloor, & \frac{n}{4} \leq|i-j|<m \\ \frac{n}{2}-1, & |i-j| \geq m\end{cases} \\ d_{i-j} &=d_{|i-j|}+\frac{n}{2} \cdot \mathbb{1}_{\{i-j>0\}} \end{aligned} \] 其中，\(a_{ij}\) 为 attention 系数，\(r_{i-j}\) 为 gated relative position bias，\(\sigma\) 为 sigmoid 函数，\(w\) 可学习，\(n=320\)，\(m=800\)。

import numpy as np
import matplotlib.pyplot as plt

n = 320
m = 800

def d_ij(i_minus_j, n, m):
    abs_diff = np.abs(i_minus_j)
    if abs_diff < n / 4:
        d_abs = abs_diff
    elif n / 4 <= abs_diff < m:
        d_abs = np.floor(n / 4 * (np.log10(abs_diff) - np.log10(n / 4)) / (np.log10(m) - np.log10(n / 4)) + 1)
    else:
        d_abs = n / 2 - 1
    return d_abs + (n / 2) * (i_minus_j > 0)

# Values for i-j
i_minus_j_values = np.arange(-m-100, m+100, 1)
d_values = [d_ij(x, n, m) for x in i_minus_j_values]

# Plot
plt.figure(figsize=(10, 6))
plt.plot(i_minus_j_values, d_values)
plt.xlabel('$i-j$')
plt.ylabel('$d_{i-j}$')
plt.grid(True)
plt.show()

3. 模拟带噪、重叠语音

随机选取噪声音频（取自 Deep Noise Suppression, DNS）或同一 batch 的语音，根据能量比叠加，重叠区域不大于原音频的 50%。信噪比 \(r\) 取值范围：噪声 [-5, 20]、重叠语音 [-5,5]。 \[ \begin{aligned} E^{\text {pri }} &\leftarrow \frac{\sum \mathbf{u}^{\text {pri }} \cdot \mathbf{u}^{\text {pri }}}{L} \\ scl &\leftarrow \sqrt{\frac{E^{\mathrm{pri}}}{10^{\frac{r}{10}} E^{\mathrm{sec}}}} \\ \mathbf{u}^{\mathrm{pri}}\left[s^{\mathrm{pri}}: s^{\mathrm{pri}}+l\right] &\leftarrow \mathbf{u}^{\mathrm{pri}}\left[s^{\mathrm{pri}}: s^{\mathrm{pri}}+l\right]+s c l \cdot \mathbf{u}^{\mathrm{sec}}\left[s^{\mathrm{sec}}: s^{\mathrm{sec}}+l\right] \end{aligned} \]

4. 训练 label

  • WavLM Base：采用 HuBERT Base 模型第一次迭代 transformer 第6层的输出，聚类。
  • WavLM Base+、WavLM Large：采用 HuBERT Base 模型第一次迭代 transformer 第9层的输出，聚类。
  • 虽然 WavLM 第一轮迭代的模型优于 HuBERT，仍采用 HuBERT 生成的伪标签可得到更好的 WavLM 第二轮迭代模型。
  • 输入音频已被破坏，为了使输出表示降噪后、主说话人的语音，借助另一个模型基于原始音频生成伪标签？

5. 评价

  • 超参数：带噪、重叠语音概率 20%，其中重叠语音占比，WavLM Base 100%，Base+、Large 模型 90%。
  • 下游任务语音表示：预训练模型各层隐藏状态的加权和，权重可学习。
    • 分析各层表示的重要性时，用对应层隐藏状态的值对权重归一化，以消除隐藏状态值的大小引入的权重 bias。

6. 其它

  • 相关工作
    • 基于训练目标，SSL 方法可以分为生成式、判别式、多任务学习。
      • 生成式：自编码器重建语音；自回归生成；非自回归模型恢复 mask 帧。
      • 判别式：wav2vec 系列采用对比损失；HuBERT 预测掩码区域的离散目标；w2v-BERT 以端到端的方式结合上述对比损失、掩码预测损失。
      • 多任务：PASE [43]、PASE+ [44] 采用多个目标，如波形生成、韵律回归、对比损失；UniSpeech [7]、JUST [45] 结合 SSL 和 ASR 监督训练。
  • 解决 fp16 数值溢出问题，主要原因为 attention score \(\frac{\mathbf{q}_i \cdot \mathbf{k}_j}{\sqrt{d}}\) 超出了 fp16 的上限，表现为 loss 为 NaN。 \[ \begin{aligned} \alpha_{i, j} & \propto \exp \left\{\frac{\mathbf{q}_i \cdot \mathbf{k}_j}{\sqrt{d}}+r_{i-j}\right\} \\ & =\exp \left\{\left(\frac{\mathbf{q}_i}{c \sqrt{d}} \cdot \mathbf{k}_j-\max _{j^{\prime} \leq T}\left(\frac{\mathbf{q}_i}{c \sqrt{d}} \cdot \mathbf{k}_{j^{\prime}}\right)\right) \times c+r_{i-j}\right\} \end{aligned} \] 其中，\(c\) 为缩放系数超参数，本文取 32。
