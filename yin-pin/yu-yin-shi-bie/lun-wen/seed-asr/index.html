Seed-ASR

发布日期:   2024-07-27
更新日期:   2024-10-29
  • Bai Y, Chen J, Chen J, et al. Seed-ASR: Understanding Diverse Speech and Contexts with LLM-based Speech Recognition[J]. arXiv preprint arXiv:2407.04675, 2024.
  • https://bytedancespeech.github.io/seedasr_tech_report
  • 作者：字节 Seed 组

1. 系统结构

2. 音频 encoder

  • 取名 LUISE: Large-scale Unsupervised Iterative Speech Encoder。

  • mel-filterbank 特征。

  • 基于 conformer。参数量：2B。

  • 离散 label

    iterative fixed。第 1 次迭代：codebook 随机初始化，采用随机投影层 [10]。之后：中间层表示 K-means 聚类。

    取第 25/32 层的表示。

  • 只计算 mask 帧的 CE loss。

  • 预训练后，取其中间层的连续表示。

3. converter

LUISE 中间层表示帧率为 40ms。拼接4个连续帧进行下采样，再输入线性投影层。因而输入 LLM 的帧率为 160ms。

4. 训练

  • 音频 encoder 自监督学习
  • LLM
    • Seed-ASR (CN)：采用超过 10B 的 MoE LLM 初始化。
  • SFT
    • learnable audio encoder + learnable converter + fixed LLM
  • 上下文 SFT
    • 数据：<上下文 ccc, 语音 xxx, 文本 yyy> + 一定比例的通用 ASR 数据。
      • 可以根据场景定制，如对话历史、视频字幕场景下的用户编辑历史、会议场景下的参会人名称。
      • 上下文由内部 LLM 根据转写文本生成，实验表明相较于采用长语音的历史转写，性能更好。
    • 解码策略优化。采用原生 beam search 存在严重的幻觉问题。

      • 剪枝：先采用 P(y∣x)P(\boldsymbol{y} \mid \boldsymbol{x})P(y∣x) 过滤声学不合理的候选 token。

        说明此处的幻觉问题是上下文引入的，P(y∣x)P(\boldsymbol{y} \mid \boldsymbol{x})P(y∣x) 只存在音准字不准的问题。

      • Joint beam search：$$P_{\text {joint }}(\boldsymbol{y} \mid \boldsymbol{x}, \boldsymbol{c})=\frac{\alpha}{1+\alpha} * P(\boldsymbol{y} \mid \boldsymbol{x}, \boldsymbol{c})+\frac{1}{1+\alpha} * P(\boldsymbol{y} \mid \boldsymbol{x})$$

  • 强化学习
    • 训练集：数千小时，部分数据含 context。
    • 由于 SFT 采用的 CE 目标函数与 CER / WER 不匹配，采用最小词错误率（minimum word error rate，MWER），同时对关键词进行加权。$$\mathcal{L}{\text {mwer }}^{\mathrm{N} \text {-best }}\left(\boldsymbol{x}, \boldsymbol{y}^*\right)=\frac{1}{N} \sum{\left.\boldsymbol{y}{\boldsymbol{i}} \in \text { N-best(x, N }\right)} \hat{P}\left(\boldsymbol{y}{\boldsymbol{i}} \mid \boldsymbol{x}\right)\left(\mathcal{W}\left(\boldsymbol{y}{\boldsymbol{i}}, \boldsymbol{y}^*\right)-\bar{W}\right)+\lambda \mathcal{L}{\mathrm{CE}}$$ $$\hat{P}\left(\boldsymbol{y}{\boldsymbol{i}} \mid \boldsymbol{x}\right)=\frac{P\left(\boldsymbol{y}{\boldsymbol{i}} \mid \boldsymbol{x}\right)}{\sum_{\boldsymbol{y}{\boldsymbol{i}} \in \mathrm{N}-\mathrm{best}(\mathrm{N}, \mathrm{N})} P\left(\boldsymbol{y}{\boldsymbol{i}} \mid \boldsymbol{x}\right)}$$ 其中 y∗\boldsymbol{y}^*y∗ 为 ground truth，W(yi,y∗)\mathcal{W}\left(\boldsymbol{y}_{\boldsymbol{i}}, \boldsymbol{y}^*\right)W(yi​,y∗) 为 （加权的）WER。
    • 为了提高 RL 训练效率，采用远程服务生成识别结果，同时在当前服务计算 loss、更新模型。

5. 模型

  • Seed-ASR (CN): 支持普通话 + 13 种方言。

    方言采用发音一致或相近的中文字人工标注。

  • Seed-ASR (ML): 多语种。

7. 实验

  • 中文 ASR 基线之一：transducer-based end-to-end model
    • MoE encoder，参数量超过 300M。
    • 在有对话上下文的场景下，采用 context FST biasing 提高关键词召回率。
