Pengi

LLM 音频理解 论文
发布日期:   2023-12-01
更新日期:   2024-07-01
  • 作者：微软、CMU
  • 创新点：将所有音频任务构建为文本生成任务：输入音频、文本，输出文本。支持封闭式和开放式音频任务。

1. 模型结构

  • 学习 HTSAT (hierarchical token-semantic audio transformer) audio encoder、mapping 网络，冻结 CLIP text encoder、自回归 LLM GPT2-base (124M)。
  • 音频：44.1kHz，窗长 1024，hop 320，64 FBank (50-8000Hz)。切为 7s 长度（约1000帧）。
  • text encoder：最长 40。
  • mapping 网络：输出固定长度（40）的 embedding 序列。
  • 解码：beam size 5。

2. 训练

  • 任务模版

  • 训练集：340w 样本。

  • 评价

    • 封闭式任务：Pengi 的输出文本、指定测试集上的所有 label 分别计算 text embedding，计算 cosine 相似度，选取最可能的类别。

  • 训练集、测试集、各任务评价指标、效果 详见论文。

3. 其它
