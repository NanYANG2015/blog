Qwen2-Audio

LLM 音频理解 论文
发布日期:   2024-07-21
更新日期:   2024-07-21
  • 优化

    • 对话能力，支持输入音频或文本，输出文本。指令微调 + DPO。

      SFT 数据的质量、复杂度严重影响模型性能。

    • 扩大训练数据量。

    • 预训练阶段，复杂的结构化的任务描述 special tokens -> 自然语言任务提示。减小预训练、指令微调阶段的 gap。

      We find that using language prompts can improve better generalization ability and better instruction following ability.

    • 音频 encoder Whisper Large-V2 -> Whisper Large-V3。LLM Qwen-7B 不变。
